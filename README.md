# ALIGN-AI-Alumni-Mentorship-App

ALIGN is an AI-driven mentorship matching platform built using Google AppSheet.
It connects students with relevant alumni mentors based on interests, skills, and help needed.

## Problem Statement
Students often struggle to find the right mentors for guidance in internships, projects, and career planning.
ALIGN bridges this gap by intelligently matching students with alumni mentors.

## Features
- Student & Alumni role-based login
- AI-based mentor matching
- Mentorship request system
- Accept / Reject requests
- Secure, user-specific data visibility
- In-app chat support (AppSheet)

## Tech Stack
- Google AppSheet (Frontend + Logic)
- Google Sheets (Database)
- AppSheet Expressions (USEREMAIL, REF, Slices)
- GitHub (Documentation & Versioning)

## AI Matching Logic
Matching is done based on:
- Student Interests
- Student Skills
- Help Needed
- Alumni Skills & Industry

Implemented using:
- AppSheet REF relationships
- Dynamic filtering
- Role-based slices

## Modules
- Student Profile
- Alumni Profile
- Find Mentors
- Mentorship Requests
- Request Status Tracking
